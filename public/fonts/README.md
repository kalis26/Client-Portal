# Licensed font files

Add the licensed WOFF2 files here before deployment:

- `CalSans-SemiBold.woff2`
- `CalSans-Regular.woff2`
- `Gotham-Book.woff2`
- `Gotham-Medium.woff2`

The app has a safe system-font fallback until those files are supplied. Do not commit commercial font files to a public repository.
